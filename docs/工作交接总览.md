## 五、AI 简报数据架构（2026-03-17 重大升级）

### 数据流

```
6个高质量 RSS 源（AIbase / 36氪AI / TechCrunch / The Verge / MIT TR / VentureBeat）
  └→ GitHub Actions 每天 08:00 触发（北京时间）
       └→ scripts/fetch-ai-news.js 抓取 RSS 标题+摘要
            └→ 调用 DeepSeek API 做分类/打标签/提炼摘要（不让它编造内容）
                 └→ 输出 ai-news-data.json
                      └→ ai-news-dashboard.html 启动时自动 fetch 加载
                           └→ 加载失败自动降级用内置数据，不会白屏
```

### 关键文件

| 文件 | 作用 |
|------|------|
| `scripts/fetch-ai-news.js` | RSS 抓取 + DeepSeek 整理脚本 |
| `scripts/fetch-news.js` | 每日要闻脚本（原有，不变） |
| `.github/workflows/news.yml` | 定时任务，两个 job 顺序执行（fetch-news → fetch-ai-news） |
| `ai-news-data.json` | AI 简报数据，每天自动生成 |
| `news-data.json` | 每日要闻数据，每天自动生成 |

### GitHub Secret

| Key | 用途 |
|-----|------|
| `DSAPIKEY` | DeepSeek API Key，两个脚本共用 |
